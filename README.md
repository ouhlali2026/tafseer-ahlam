# موقع تفسير الأحلام الشامل 🌙

مرجعك الأول والموثوق لتفسير أحلامك وفق أشهر المفسرين (ابن سيرين، النابلسي، ابن شاهين) مع إمكانية مشاركة الأحلام وتبادل التجارب.

![موقع تفسير الأحلام](https://via.placeholder.com/1200x600/2C3E50/FFFFFF?text=تفسير+الأحلام)

## ✨ المميزات

- **تفسيرات دقيقة**: مستمدة من التراث الإسلامي مع إعادة صياغة بأسلوب معاصر.
- **تصنيفات متعددة**: حسب الحروف الهجائية، الموضوعات، والمفسرين.
- **مشاركة الأحلام**: يمكن للمستخدمين مشاركة أحلامهم وقراءة أحلام الآخرين.
- **بحث سريع**: ابحث عن تفسير أي حلم بسهولة.
- **تصميم متجاوب**: يعمل على جميع الأجهزة (هواتف، أجهزة لوحية، حواسيب).
- **سرعة فائقة**: موقع ثابت (Static) مع تحميل ديناميكي للبيانات.

## 🛠️ التقنيات المستخدمة

- **HTML5**: هيكل الموقع.
- **CSS3**: التصميم والتنسيق (مع خط Tajawal).
- **JavaScript**: تحميل البيانات وعرضها ديناميكياً.
- **JSON**: تخزين جميع التفسيرات بشكل منظم.
- **GitHub Pages / Vercel**: للنشر والاستضافة.
